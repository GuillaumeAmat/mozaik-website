Mozaïk comes with 5 themes, you can set the theme you want with the `theme` key in the config file:

``` javascript
{
  //…
  theme: 'yellow',
  //…
}
```
## Available themes

- bordeau
- light-grey
- yellow
- light-yellow
- night-blue
